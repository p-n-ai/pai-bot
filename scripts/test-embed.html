P&AI Embed Widget Test

This page tests the embeddable web chat widget. Look for the chat bubble in the bottom-right corner.

How it works:
1. The <script> tag below loads /embed/pai-chat.js
2. The widget creates a toggle button and an iframe
3. Click the bubble to open the chat
4. The iframe fetches a guest token and connects via WebSocket

Tenant: default
Color: #6366f1 (indigo)
