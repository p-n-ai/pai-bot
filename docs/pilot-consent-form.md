# Pilot Consent Form (Student/Parent)

## Purpose
You are invited to test a KSSM Matematik AI tutor (Form 1-3 Algebra). This pilot uses a new channel (Telegram). The goal is to improve teaching quality and detect errors.

## What We Collect
- Chat messages with the bot
- Time and usage metadata (timestamps, session counts)
- Feedback form responses

## Storage
Data is stored on AWS for review and improvement purposes.

## Access and Use
Only the project team will review the chat logs to evaluate correctness, clarity, and tutoring behavior.

## Voluntary Participation
Participation is voluntary. You may stop at any time. We will stop using your data upon request.

## Age and Parent Consent
Participants are existing Pandai users, but if the participant is under 18, parent or guardian consent is required before access. This applies even though the student is already in the Pandai system because the pilot uses Telegram and includes chat log review.

## Incentive
Tokens are provided after completing the feedback form.

## Contact
If you have questions or want to withdraw, contact the project team.
