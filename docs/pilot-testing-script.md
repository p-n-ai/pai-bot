# Pilot Testing Script (KSSM Form 1-3 Algebra)

## Scope
Algebra topics only (Form 1-3). Avoid calculus, non-algebra topics, and advanced university math.

## Channels
- Primary testing channel: Telegram
- Optional coordination: Discord

## Start Message to Volunteer
Please try 6 to 10 questions. Use the prompts below. Answer normally. If the bot gives a full answer too fast, note it in the feedback form.

## Safe Prompt List
1. Solve 3x - 5 = 16. Ask for the first step only.
2. Form an equation: "A number plus 7 is 19."
3. Complete the sequence: 7, 13, __, 25, __.
4. Factorise x^2 + 6x + 8.
5. Simplify (x + 2)(x - 5).
6. Translate to algebra: "Ali has 3 more marbles than Siti."
7. Verify: "I got x = 5 for 5x - 7 = 18. Check only."
8. A taxi charges RM4 base + RM2 per km. Write the equation for 5 km.
9. Ask for a real-life analogy for 2x = 10.
10. Ask for the next term in a geometric sequence (e.g. 1, 0.3, 0.09, ...).

## Prompt Tambahan (Fungsi Baharu)
1. Tetapkan matlamat: "/goal Saya nak kuasai Algebra Tingkatan 1 dalam 2 minggu."
2. Semak kemajuan: "/progress"
3. Minta kuiz: "Saya nak kuiz Algebra Tingkatan 1." (kuiz boleh bermula melalui ayat biasa tanpa arahan khas)
4. Ubah matlamat: "/goal Saya nak fokus Pola dan Jujukan minggu ini."
5. Padam matlamat: "/goal clear"

## What to Avoid
- Integration or differentiation
- Non-algebra topics (e.g. statistics, geometry unless related to algebra)
- SPM-style proofs beyond Form 3 algebra

## Expected Behavior
- The bot should not dump the full answer immediately when asked for "first step only" or "check only".
- The bot should stay within KSSM Form 1-3 Algebra.
- The bot should be clear, supportive, and step-by-step.

## After Testing
Complete the feedback form and submit.
